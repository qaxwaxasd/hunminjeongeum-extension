# 프로젝트: 한국어 맞춤법 교정 확장프로그램 (그레머리 KR)

본 프로젝트는 인공지능 입문 강의 산출물로, '그레머리(Grammarly)'의 핵심 기능을 벤치마킹하여 실시간 맞춤법 및 문법 교정을 제공하는 크롬 확장프로그램입니다.

## 1. 프로젝트 개요
- **모델:** 그레머리(Grammarly) 크롬 확장프로그램
- **핵심 목표:** 실시간 오타 및 문법 오류 수정, 정확한 문장 부호 제안
- **디자인 컨셉:** 초록색 포인트 테마 (다크/라이트 모드 지원)

## 2. 디자인 시스템 (Point Color)
가독성과 긍정적인 피드백을 위해 초록색 계열의 포인트 컬러를 사용합니다.

| 항목 | 다크 모드 (Dark) | 라이트 모드 (Light) | 설명 |
| :--- | :--- | :--- | :--- |
| **Primary (Point)** | `#52C41A` | `#389E0D` | 부드럽고 생동감 있는 초록 |
| **Primary-Hover** | `#73D13D` | `#237804` | 버튼 호버 상태 |
| **Sub-Highlight** | `#B7EB8F` (20% Opacity) | `#F6FFED` | 텍스트 배경 강조 |
| **Base BG** | `#111827` | `#F9FAFB` | 전체 배경 |
| **Error Line** | `#FF4D4F` | `#CF1322` | 오타 밑줄 (초록과 대비) |

## 3. 기능 요구사항 (Functional Requirements)
| ID | 기능 핵심 | 상세 가이드 |
| :--- | :--- | :--- |
| **FR-01** | 실시간 검사 | 웹 페이지 입력창에서 실시간 맞춤법/문법 검증 수행 |
| **FR-02** | 문맥 구두점 제안 | 쉼표, 마침표 등 문장 부호 위치 제안 |
| **FR-03** | 오류 하이라이트 | 오류 발견 시 밑줄(오타: 빨강, 문법: 초록/파랑 점선) 표시 |
| **FR-04** | 교정 카드 제공 | 하이라이트 클릭 시 상세 교정 이유와 대안 단어 표시 |
| **FR-05** | 즉시 수정 | 교정안 클릭 시 원문을 즉시 대안 단어로 교체 |
| **FR-06** | 서비스 토글 | 확장 프로그램 팝업에서 서비스 활성화/비활성화 제어 |
| **FR-07** | 제외 사이트 등록 | 코딩 사이트 등 특정 도메인을 검사 대상에서 제외 |
| **FR-08** | 클립보드 복사 | 교정 완료된 전체 텍스트를 한 번에 복사 |
| **FR-09** | 오류 통계 확인 | 맞춤법/띄어쓰기/문법 유형별 통계 대시보드 제공 |

## 4. 비기능 요구사항 (Non-Functional Requirements)
- **성능:** 입력 중단 후 **2초 이내** 결과 표시, 타이핑 렉(Lag) 방지 (비동기 처리)
- **사용성:** 원문을 가리지 않는 동적 팝업 배치, 시스템 모드(Dark/Light) 자동 대응
- **보안:** 모든 전송은 **HTTPS** 암호화, 비밀번호 필드 검사 금지, 데이터는 교정 후 **즉시 파기**
- **자원 효율성:** 메모리 점유율 **100MB 이하**, CPU 점유율 **5% 미만** 유지

## 5. 구현 가이드 (Developer Notes)
- **UI:** 설계서에 명시된 초록색 포인트 테마를 CSS 변수 또는 Tailwind Config에 적용할 것.
- **Backend:** 데이터 휘발성 원칙에 따라 서버 메모리에서 즉시 삭제 로직 구현 필수.
- **Compatibility:** Chromium 기반 브라우저(Chrome, Edge, Whale) 최신 버전 지원.
